待宵物語 ブラウザゲーム初回再構築版 v0.1

■ 内容
このZIPは、GitHub Pagesへ再配置しやすいように作り直した静的構成の試作版です。
一枚絵マップ依存をやめ、タイル描画ベースへ切り替えています。

■ この版で入っているもの
- タイトル画面
- はじめから / つづきから
- コルパンの家から開始
- 母のメモを読む
- 外へ出る
- 村を歩く
- ルルガーの家へ入る
- ルルガーの前まで行く
- 会話中の十字キー無効化

■ 構成
- index.html
- style.css
- game.js
- assets/

■ 使い方
1. ZIPを展開
2. 中身をそのままGitHubの公開用リポジトリへ配置
3. GitHub Pagesを有効化
