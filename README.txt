ガーガル村 試作版
=================

内容
----
スマホ向けブラウザRPGの試作版です。
以下の導入イベントを遊べます。

- コルパンの家で母のメモを読む
- ルルガーの家で昼食イベント
- タート来訪と配達開始
- メル合流
- 村内配達
- モガモ宅で手紙の異変
- 村はずれの墓地でミガムの墓を確認
- 野雉っぽい鳥との初戦闘
- ルルガーへの報告

ファイル
--------
- index.html
- style.css
- game.js
- assets/ （画像素材）

遊び方
------
1. index.html をブラウザで開く
2. スマホでは画面下の方向ボタンで移動
3. 光っている枠の近くで「調べる」を押す
4. 会話を読み進める

GitHub Pages で公開する方法
---------------------------
1. このフォルダの中身を GitHub リポジトリへアップロード
2. Settings > Pages を開く
3. Deploy from a branch を選ぶ
4. Branch を main / root にする
5. 数分待つと公開URLが出ます

補足
----
- これは試作版なので、NPC常時表示や本格的な当たり判定はまだ簡略化しています
- 画像の上に青いガイド枠が出ますが、試作版では操作しやすくするために残しています
