# 待宵物語 一画面プロトタイプ v0.3

## 目的

このフォルダは、第一章プロトタイプの最初の実装確認用です。

最初の確認対象は以下です。

1. コルパンが村の一画面を歩ける
2. ランドが同じ画面に存在する
3. NPCに話しかけられる
4. 会話ウィンドウと選択肢が表示される
5. ルルガー家前まで移動できる

## 起動方法

`index.html` をブラウザで開いてください。

ただし、環境によっては `fetch()` でローカルJSONを読めない場合があります。
その場合は、このフォルダで簡易サーバーを起動してください。

```bash
python3 -m http.server 8000
```

その後、ブラウザで以下を開きます。

```text
http://localhost:8000
```

## 操作

- 矢印キー / WASD：移動
- Enter / Space：決定
- 画面下の十字キー：移動
- 決定ボタン：会話

## 現段階の仕様

- マップは固定一画面
- コルパンは4コマ仮アニメ
- ランドはプレイヤーの旧位置へ簡易追従
- NPCはルルガーの仮配置
- 会話UIはCSSベース
- 背景はコード描画ベース
- 仮切り出し素材はキャラとUIを中心に使用

## 注意

このプロトタイプは完成画面ではありません。

目的は、以下を早く確認することです。

- 縮尺
- 視認性
- 操作感
- 会話UIの読みやすさ
- スマホ縦画面での余白
