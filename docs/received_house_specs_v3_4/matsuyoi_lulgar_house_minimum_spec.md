# 『待宵物語』ルルガー家 外観・内観 最小仕様書

**作成日**: 2026-04-24  
**担当**: グラフィック部門  
**用途**: 第一章プロトタイプ向け / 実装部門引き渡し用

---

## 1. この仕様書の目的

この文書は、第一章前半の中心舞台となる**ルルガー家**について、  
外観・内観の最小構成を、実装部門がそのまま組める粒度で定義したものである。

ルルガー家は、単なる家ではなく以下の三要素を同時に満たす必要がある。

- 村長宅である
- 家庭の場である
- 郵便仕分けや来客対応が行われる場である

目的は、豪華な完成状態を先に目指すことではなく、  
**「人が集まる家」「村長の家」「生活がある家」** を最小構成で成立させることである。

---

## 2. 空間の基本方針

ルルガー家は、コルパン家より**少し広く、少し整っている**。  
ただし、豪邸や役所のようにはしない。

### 第一章で優先する印象
1. 村長の家である
2. しかし家庭の場でもある
3. 昼食や会話が自然に行える
4. 郵便束や簡易な文書が扱える
5. 来客があっても不自然でない

### 避けること
- 城館のような豪奢さ
- 学者の家のような書物の山
- 兵士の詰所のような硬さ
- 権威だけを感じる無機質さ
- 家庭感が薄い「役所化」

---

## 3. 外観の最小構成

以下を**最小構成**とする。

1. 石壁または整った外壁
2. 屋根
3. 扉
4. 窓
5. 入口前の余白
6. 村長宅らしい整い方
7. 郵便配達や来客が立ち寄って不自然でない前面空間

### 外観の方針
- コルパン家より少し立派
- ただし村の中で浮かない
- 「目立ちすぎない中心性」がある
- 入口まわりは、来訪導線が分かりやすいこと

---

## 4. 内観の最小構成

以下を**最小構成**とする。

1. 床
2. 内壁
3. 扉
4. 食卓
5. 椅子
6. 調理の気配
7. 村長としての仕事机または文書置き場
8. 郵便束を扱える場所
9. 収納
10. 来客がいても不自然でない余白

---

## 5. 外観レイアウト方針

### 5-1. 家の見え方
- 一目で「家」と分かる
- ただし村の中心的人物の家として、やや整って見える
- 屋根・壁・扉の視認性を優先する

### 5-2. 入口前
- NPCが立てる余白を確保
- タートが来訪しても窮屈に見えない
- コルパンとランドが接近しても崩れにくい

### 5-3. 付帯表現
- 家前に小物を置きすぎない
- 看板のような強い記号は不要
- 村長宅らしさは、整い方と余白で示す

---

## 6. 内観レイアウト方針

### 6-1. 中心
- 食卓を中心に置く
- 「昼食の場」として自然な見え方を優先する

### 6-2. 壁側
- 収納、棚、調理設備を寄せる
- 文書や郵便束を扱える場所は机まわりへ寄せる

### 6-3. 村長の仕事感
- 仕事机は大げさにしない
- 村の記録や簡易文書を扱う程度で十分
- 家庭空間と仕事空間が完全に分離しない方がよい

### 6-4. 来客導線
- 入ってすぐ家具に詰まらない
- 会話イベントが起きても窮屈に見えない
- コルパン、ルルガー、タート、ランドがいても破綻しにくい余白を意識する

---

## 7. 家具ごとの役割

### 食卓
ルルガー家の中心。  
家庭性を支えると同時に、第一章の会話空間として重要。

### 椅子
食卓と対で置く。  
来客があっても成立する程度に数を持たせてもよいが、最初は最小限でよい。

### 調理の気配
ルルガーが料理もする人物であることを支える。  
かまど、作業台、鍋、簡素な調理器具で十分。

### 仕事机 / 文書置き場
村長としての役割を支える。  
ただし「役所」ではなく「家の中にある仕事の場」として見せる。

### 郵便束置き場
第一章上、重要。  
タート来訪や仕分けの導線に自然につながる位置が望ましい。

### 収納
生活感の補強。  
書類棚ではなく、まずは日用品や食器も含めた生活収納として扱う。

### 次点

- lamp
- crate

- additional document spot

### 後回し

- 来客用の細かな演出小物
- 第二章以降を見越した追加部屋

## 9. collision 方針

### 基本
- 外壁: true
- 屋根: true
- 扉: true
- 窓: true
- 床: false
- 食卓: true
- 椅子: true
- かまど: true
- 作業台: true
- 郵便束: true
- 収納: true
- 小さな卓上小物: false

### 運用
基本は tile collision を持たせ、必要なら collision layer で上書きする。

---

## 10. 外観の最小成立条件

- 村の中でルルガー家だと認識しやすい
- 入口導線が分かる
- コルパン家より少し整って見える
- 来客イベントが起きても不自然でない
- 村長宅だが、豪奢ではない

---

## 11. 内観の最小成立条件

- 食卓があり、家庭空間として成立している
- 調理の気配がある
- 村長の仕事感が少しある
- 郵便束を扱う場所がある
- 人が集まれる家として見える
- しかし村の家の範囲から逸脱しない

---

## 12. 次段階で追加する候補

- 食卓上の食器差分
- 村の記録物差分
- 郵便仕分け専用台
- 来客対応用の座席差分
- 調理器具差分
- 収納の密度差分
- 第二室または奥側空間の追加

---

以上を、ルルガー家 外観・内観 最小仕様の正式実装仕様とする。
