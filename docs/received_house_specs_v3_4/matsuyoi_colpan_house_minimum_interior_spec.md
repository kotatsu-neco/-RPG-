# 『待宵物語』コルパン家 最小内観セット 実装仕様書

**作成日**: 2026-04-24  
**担当**: グラフィック部門  
**用途**: 第一章プロトタイプ向け / 実装部門引き渡し用

---

## 1. この仕様書の目的

この文書は、第一章冒頭で使用する**コルパン家の最小内観セット**を、実装部門がそのまま組める粒度で定義したものである。  
目的は、豪華な完成状態を先に目指すことではなく、以下の体験を最小構成で成立させることである。

- コルパンの家に入れる
- 「帰る家」であることが分かる
- 母のメモの存在が自然に置かれている
- 生活感がある
- しかし裕福すぎず、寂れすぎもしない

---

## 2. 空間の基本方針

コルパン家は、**質素だが寂れさせない**ことが重要である。  
家具を多く置いて豪華に見せるのではなく、**必要なものがある静かな生活空間**として成立させる。

### 第一章で優先する印象
1. 帰る家である
2. 母の不在が静かに分かる
3. コルパンが生活者である
4. ランドの存在が不自然でない
5. 生活のぬくもりが完全には消えていない

### 避けること
- 村長宅のような広さ
- 学者の家のような本や紙の多さ
- 旅人の拠点のような装備の山
- 貧しさを強調しすぎる寂れ表現
- 小物過多による視認性低下

---

## 3. 実装時の最小構成

以下を**最小構成**とする。

1. 木床
2. 内壁
3. 扉
4. ベッド（または簡素な寝床）
5. 机
6. 椅子
7. 棚
8. 母のメモ置き場
9. 小さな調理気配
10. 生活用収納

---

## 4. レイアウト方針

### 4-1. 空間サイズ感
- 小さめ〜中程度の家
- 1画面〜1.5画面で把握できる程度
- スマホ縦画面で見たときに、中央の家具が埋もれないこと

### 4-2. 推奨配置
- **入口付近**: 余白を少し確保し、入室時に家具へ即衝突しないようにする
- **中央付近**: 机 + 椅子  
- **壁際**: 棚、収納
- **片側**: ベッド
- **机上または机付近**: 母のメモ置き場
- **隅または壁沿い**: 小さな調理気配

### 4-3. ランドへの配慮
- ランドが家の中にいても不自然でないだけの余白を残す
- 通路を極端に狭くしない
- 家具を壁沿いへ寄せすぎず、導線を確保する

---

## 5. 家具ごとの役割

### 木床
空間の基礎。  
温かさと生活感を支える。過度に豪華な木目は不要。

### 内壁
外壁より少し明るく、圧迫感を出しすぎない。  
生活空間として落ち着けるトーンにする。

### 扉
入口として一目で分かること。  
必要なら `door_highlight` を併用する。

### ベッド
コルパンの寝床。  
豪華ではなく、**ちゃんと使われている寝床**として見えることが重要。

### 机
生活と物語の中心。  
母のメモと関係づけやすい配置にする。

### 椅子
机と対で置く。  
机だけ、椅子だけの不自然な構成は避ける。

### 棚
収納と生活感の補強。  
ただし物を詰め込みすぎない。

### 母のメモ置き場
第一章冒頭の導線上、重要。  
専用オブジェクトでなくてもよいが、**ここにメモがある**と自然に分かる場所にする。

### 小さな調理気配
火を使う生活の痕跡として必要。  
かまど、鍋、簡素な作業台などで十分。

### 生活用収納
衣類や日用品の収納を思わせる。  
タンス、大箱、収納箱などで代替可。

## 7. collision 方針

### 基本
- 床: false
- 壁: true
- 扉: true（入口制御は別管理でも可）
- ベッド: true
- 机: true
- 椅子: true
- 棚: true
- メモ置き場: false
- 調理気配: true / false はオブジェクトにより調整
- 収納: true

### 運用
基本は tile collision を持たせ、必要なら collision layer で上書きする。

---

## 8. 実装部門への引き渡し時の注意

- 最初は**完全な部屋の豪華さ**ではなく、**導線と印象**を優先する
- 母のメモは、最初のイベント導線に自然につながる位置へ置く
- ランドがいる場合の移動余白を先に検証する
- 小物は後から足せるので、まずは床・壁・机・椅子・ベッド・棚を成立させる
- 実機確認後に、配置密度と当たり判定を微調整する

---

## 9. 最小成立条件

コルパン家のプロトタイプは、以下が満たされれば成立とみなす。

- 家の入口が分かる
- 中に入れる
- 木床と内壁で室内が成立している
- ベッド、机、椅子、棚のうち主要4点が見える
- 母のメモの位置が分かる
- コルパンが移動しても窮屈すぎない
- 家の印象が「質素だが帰る家」である
