# ストーリー部門・グラフィック部門への確認事項

## ストーリー部門へ確認したいこと

### 1. 第一章で必要なシーン一覧

現時点で想定している候補:

- ガーガル村 中心部
- ルルガー家
- コルパンの家
- ミガムの家
- 村道
- 墓地
- 野雉との初戦エリア

第一章で最低限必要なマップ単位を確定したい。

### 2. 第一章の進行フラグ

例:

- 母のメモを読んだ
- ルルガーに会った
- タートが来た
- 手紙を受け取った
- ミガムに届けた
- 墓地へ行ける
- 野雉戦が発生した
- 報告した

これを正式な順序で整理したい。

### 3. 手紙・母のメモの扱い

- アイテムとして持つのか
- イベントログ扱いにするのか
- メニューの「アイテム」に表示するのか
- 「大事なもの」カテゴリを分けるのか

## グラフィック部門へ確認したいこと

### 1. 正式タイルサイズ

現状は 16x16 タイルを前提。

- 16x16で確定するか
- 16x16を基準に、オブジェクトは複数タイルで組むか
- 家屋や木などは大判オブジェクトPNGにするか

### 2. 最初に必要なタイルセット

最低限必要:

- 草地
- 暗い草地
- 土道
- 石畳
- 石壁
- 屋根
- 木材
- 畑
- 井戸
- 入口ハイライトまたは入口表現
- 屋内床
- 屋内壁
- 棚
- テーブル
- かまど / 作業台

### 3. ルルガー専用スプライト

現状は仮NPC画像。

- 正式ルルガーの立ち絵 / 歩行スプライトが必要
- まずは正面1枚でもよいか
- 4方向歩行まで必要か

### 4. コルパン・ランドの方向別スプライト

現在は仮4枚を流用。

- front / back / left / right
- 各方向2〜4フレーム
- ランドは大型犬らしさを維持する

## 実装側からの提案

次に必要なのは、ストーリー全部ではなく、まず「第一章最小マップ一覧」と「正式タイルセット最小仕様」です。
