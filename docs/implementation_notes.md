# 実装メモ

## 現在の判断

今回は、仮切り出し素材のうちキャラクターを先に使い、背景はコードで簡易描画しています。

理由は以下です。

- 村タイル候補はまだ用途名が確定していない
- そのまま敷き詰めると縮尺検証より素材分類作業が先行する
- まず「歩ける・話せる・見やすい」を成立させる必要がある

## 次に実装すべきこと

1. コルパンの方向割当
   - seq01〜seq04 を front / back / left / right に確定する
2. ランドの方向割当
   - 大型犬らしく見える候補を4方向に確定する
3. UI画像の実使用
   - 現在はCSS窓
   - 次段階で `assets/ui/` の候補を9スライス化
4. 背景タイルマップ化
   - 現在はコード描画
   - 次段階で `assets/tiles/village_minimum/` を分類し、タイルID方式へ移行
5. ルルガー家前イベント
   - 入口タイルに接触したときの通知・遷移を追加

## 実装上の注意

- スマホ縦画面では下部UIと会話窓が干渉しやすい
- 会話中は移動を止める
- 背景の密度を上げるより、導線と視認性を優先する
- ランドは小型犬に見えないよう24x24基準を維持する
